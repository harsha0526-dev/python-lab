10. File Processing
Start the program and define function for processing file.
Initialize variables to count lines, words, and long words.
Open input file in read mode with proper encoding.
Read file line by line using loop structure.
Increment line counter for each line read from file.
Split each line into words and count total words.
Check each word length and collect words longer than five.
Store long words in a separate list for output purpose.
Write collected long words into output file line by line.
End the program after completing file processing successfully.

def process_file(input_file, output_file):
    line_count = 0
    word_count = 0
    long_words = []

    # Read the input file
    with open(input_file, 'r', encoding='utf-8') as f:
        for line in f:
            line_count += 1
            words = line.split()
            word_count += len(words)
            # Collect words longer than 5 characters
            long_words.extend([word for word in words if len(word) > 5])

    # Print counts
    print(f"Total lines: {line_count}")
    print(f"Total words: {word_count}")

    # Save long words to output file
    with open(output_file, 'w', encoding='utf-8') as f:
        for word in long_words:
            f.write(word + '\n')


# Example usage
input_file = "input.txt"   # Replace with your file name
output_file = "long_words.txt"
process_file(input_file, output_file)
