3. Factorial (Recursion + Lambda)
Start the program and import reduce function from functools module.
Define a recursive function to calculate factorial of number.
Inside function, check whether number is zero or one.
If condition is true, return one as factorial result.
Otherwise recursively call function with decremented number value.
Define lambda function using reduce to compute factorial iteratively.
Prompt user to input a number for factorial calculation.
Convert the input value into integer data type.
Display factorial using both recursion and lambda approaches.
End the program after displaying both computed results.

from functools import reduce 
 
 
def factorial_recursive(n): 
    if n == 0 or n == 1: 
        return 1 
    else: 
        return n * factorial_recursive(n - 1) 
 
 
factorial_lambda = lambda n: reduce(lambda x, y: x * y, range(1, n + 1), 1) 
 
 
def main(): 
    try: 
        num = int(input("Enter a number to compute factorial: ")) 
 
        print(f"Factorial of {num} using recursion: {factorial_recursive(num)}") 
 
        print(f"Factorial of {num} using lambda & reduce: {factorial_lambda(num)}") 
 
    except ValueError: 
        print("Invalid input! Please enter an integer.") 
 
 
if __name__ == "__main__": 
    main()
